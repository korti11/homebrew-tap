# korti11/tap

Homebrew tap for korti11's CLI tools.

## Setup

```bash
brew tap korti11/tap
```

You only need to do this once. All tools in this tap will then be available via `brew install`.

## Available formulae

| Formula | Description |
|---------|-------------|
| `satisfactory-pathfinder` | Satisfactory factory planning companion CLI |

## Installation

```bash
brew install korti11/tap/satisfactory-pathfinder
```

## Updating

```bash
brew upgrade satisfactory-pathfinder
```
